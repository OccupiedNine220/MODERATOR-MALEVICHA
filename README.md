# [RU] Модератор Малевича
#### Discord бот для модерации.

### Функции:
- Антирейд
- Модерация по Вашим правилам (НЕ РАБОТАЕТ!)
- Кланы на сервере (ТЕСТ)
- Банворды
- И другие прикольные штуки

### Установка и запуск
1. Скачайте исходный код с Git
2. Отредактируйте файл automod.py с Вашим токеном на [оффицальном портале для разработчиков Discord](https://discord.com/developers/ "Портал разработчиков Discord")
Добавьте в конце:
```python
bot.run('ТОКЕН')
```
4. Запустите бота с помощью `python3 automod.py`(на Windows `python automod.py`)
> Применчание: Бот назначен только для одного сервера.
5. Наслаждайтесь ботом!

Связь с разработчиком в `Issues` или в Discord `occupiednine220`.
